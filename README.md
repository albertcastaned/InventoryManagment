# InventoryManagment
Small Java application to handle an inventory's stock using concurrency programming paradigm.

## How to run
### Requirements
* Java Runtime Environment (JRE)
* Port 6000 not used by any other application

### Steps
1. Open a server instance with the file `Server.jar`
2. Open a or multiple client instance(s) with the file `Client.jar`
3. Do requests
